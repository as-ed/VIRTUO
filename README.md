# V.I.R.T.U.O.

## Prerequisits

* Python 3.9
* pipenv
* MPV
* Pandoc
* Tesseract OCR
* NLTK
* PicoTTS
* Google Cloud API access

## Run on Raspberry Pi

1. sync project to Raspberry Pi: `./rpi-sync <rpi-ip-addr>`
2. install dependencies: `pipenv install`
3. run project: `./run`

## Run Locally

1. install dependencies: `pipenv install --dev`
2. run project: `./run
